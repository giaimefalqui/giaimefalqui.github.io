Optimal Control Lap Time Simulation

Ecurie-Aix Formula Student Team

This project involved developing a comprehensive Optimal Control framework to minimize lap time for a 4-wheel-drive electric Formula Student vehicle. The system combines advanced control theory with vehicle dynamics simulation.

Key Contributions:
• Mathematical formulation of the optimal control problem using Pontryagin's Maximum Principle
• Integration of vehicle dynamics model with tire friction constraints
• Development of efficient numerical solver using CasADi and MATLAB
• Interactive GUI for parameter setup, simulation execution, and results visualization
• Lap time predictions with ±2-3% accuracy compared to experimental data

Results: 2nd place at the FSG MathWorks Modeling & Simulation Award 2025, recognizing the excellence in model development and practical application to vehicle performance optimization.
